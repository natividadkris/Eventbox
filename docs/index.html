Getting Started

Welcome to EventBox's Documentation!!!

Before going to the documentation of the Eventbox's modules, we must know what are the different types of user and their role in the system, and how does each module work in the system.

There are two types of user :

  • Registered User - users who have account in the system. Registered user can be a ORGANIZER or PARTICIPANT . Organizer are user who manages event and Participant are user who participate event.

  • Guest User - users who does not have account in the system but has the authority to explore in searching events.

There are currently 6 module in the system:

  • Manage Event - a module where organizer is able to create, delete, and edit event information and its participant list.

  • Create Registration Form - a module where organizer is able to create registration form.

  • Send Invitation - a module where organizer is able to invite and send invitation to the participants

  • Approve Invitation - a module where participant is able to approve or deny the invitation send by the Organizer.

  • Send Request - a module where participant is able to send request or ask permission to join event in organizer.

  • Approve Request - a module where organizer is able to approve request to join event by event registration.

Now you are brief about the different types of user and thier role in the system and also the module functionality.
